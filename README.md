# 🌟 She Can Foundation - Web Development Assignment

A premium, fully responsive single-page website built as part of the Web Development Internship application for **She Can Foundation**. This project demonstrates core frontend fundamentals blended with modern dynamic user experience elements.

## ✨ Features Implemented

### 📋 Core Requirements (Mandatory)
* **Structured Sections:** Complete layout including Hero/Home, About Us, Core Pillars, Join Us (CTA), and a clean Footer.
* **Contextual Imagery:** High-resolution professional imagery reflecting the theme of women empowerment and skill development.
* **Call to Action:** Dedicated interactive "Join Us" button to convert users into volunteers.

### 💎 Advanced Add-ons (Bonus Core)

* **Buttery Smooth Scrolling:** Integrated **Lenis JS** for a luxury, fluid scroll feel across desktop browsers.
* **Scroll-Driven Animations:** Implemented subtle text-reveals, element entry animations, and card fade-ins using **GSAP (GreenSock) & ScrollTrigger**.
* **Dynamic Statistics Counter:** Numbers automatically count up from zero when the user scrolls down to the About section.
* **Interactive Custom Cursor:** A sleek, fluid global cursor that scales up dynamically upon hovering over interactive components like links and buttons.
* **100% Fully Responsive:** Crafted with raw **CSS3 Media Queries**—completely optimized for Mobile, Tablet, and Desktop screens without relying on heavy frameworks.

---

## 🛠️ Tech Stack Used

* **Frontend Structure:** HTML5 (Semantic Tags)
* **Styling & Layout:** Core CSS3 (Custom Variables, Flexbox, Media Queries)
* **Interactions & Logic:** Vanilla JavaScript (ES6+)
* **Animation Libraries:** GSAP (GreenSock Animation Platform), ScrollTrigger
* **Smooth Scroll Engine:** Lenis JS
